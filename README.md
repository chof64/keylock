# Keylock: A Centralized Door Lock System with RFID.

Keylock is a centralized door lock system designed to enhance security and access
management using RFID technology. It incorporates a centralized control system
that allows for easy granting and revoking of access permissions, timely monitoring
of entry and exit activities, and improved logging of access events. Keylock is
also designed to be scalable, allowing for easy integration of multiple buildings
and doors into the centralized platform.

## Features

- Door Locking and Unlocking: The system allows for locking and unlocking doors using
  RFID cards or via the centralized platform.
- Centralized Control: The system provides a centralized control platform for managing
  access permissions, monitoring entry and exit activities, and logging access events.
- RFID Integration: The system uses RFID technology for secure and efficient access
  management. Create RFID cards for users and assign access permissions.
- User Management: The system allows for easy management of users and their access
  permissions. Add, remove, or modify user access permissions as needed.
